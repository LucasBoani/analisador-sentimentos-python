# Analisador de Sentimentos com Python 🤖

Este projeto foi desenvolvido como parte dos meus estudos em **Ciência da Computação no IDP**. O objetivo é aplicar conceitos de Processamento de Linguagem Natural (NLP) para classificar o sentimento de frases.

## 🚀 Tecnologias utilizadas
- Python 3.14
- Biblioteca TextBlob

## 🛠 Como rodar o projeto
1. Instale a biblioteca necessária:
   `pip install textblob`
2. Execute o script:
   `python analisador.py`

## 📊 Resultados
O script analisa polaridades e retorna se o texto é Positivo, Negativo ou Neutro utilizando emojis para facilitar a visualização no terminal.
